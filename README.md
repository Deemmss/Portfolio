- Compétences
- Contact

## Mise en ligne avec GitHub Pages

1. Créer un dépôt GitHub nommé `tonpseudo.github.io`
2. Ajouter les fichiers `index.html`, `style.css`, `script.js` et `README.md`
3. Aller dans Settings > Pages
4. Choisir la branche `main`
5. Le site sera disponible à l'adresse `https://tonpseudo.github.io`
