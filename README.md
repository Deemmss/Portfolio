# Portfolio de Soren Miguet

Portfolio personnel en HTML, CSS et JavaScript, publié avec GitHub Pages.

## Pages

- `index.html` : présentation, expériences, projets, compétences et contact
- `virbac.html` : étude de cas détaillée du projet de maintenance prédictive chez Virbac
- `assets/virbac/` : rapport et livrables consultables depuis l’étude de cas

## Publication

Le site est statique : aucun build n’est nécessaire. Il suffit de publier la branche principale avec GitHub Pages.
